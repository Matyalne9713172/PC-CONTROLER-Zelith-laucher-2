# 🎮 PC CONTROLLER (buttons) for Zaltih Launcher 2

A new, colorful PC-style controller interface for the **Zaltih Launcher 2** app.
---

## 📌 Overview

This controller adds a virtual gamepad-like button panel to the launcher, making it easier to navigate and control games or system functions directly from the interface.

Designed to be:
- 🟢 **Colorful** – easy to distinguish buttons at a glance
- 🟡 **Intuitive** – familiar layout like a classic PC gamepad
- 🔵 **Simple to use** – plug-and-play UI integration

---

## 🎨 Features
- Customizable button colors
- Quick-launch shortcuts for games or tools
- Clean, responsive layout for any window size
- Lightweight and snappy performance
