# Update Notes — V.1.8.PRE-Release

This pre-release version introduces a major overhaul of the interface and controls to improve ergonomics, combat response times, and navigation convenience.

## 🚀 Key Changes and Features:

### 🎮 Control & Combat Optimization
* **Combat Block:** Attack (`Left mouse button`) and interact (`right mouse button`) buttons have been moved to the bottom right corner for easy thumb access.
* **Movement Dynamics:** The `Shift` (crouch) and `Jump` buttons are now placed directly below the mouse buttons. This allows for seamless maneuvering while fighting.
* **Dedicated Sprint:** Added a standalone `Sprint` button on the left side of the screen for instant sprinting without extra key combinations.

### 🛠 UI & Administration:
* **Quick Mode Switch:** Integrated a `Switch gm mode` button for instant gamemode switching without typing commands in the chat.
* **Visibility Control:** Added experimental `Hide Gui` duplicate buttons in the top right corner for flexible HUD toggling.
* **Chat & Communication:** Optimized the keyboard call area via `TAB` and added a dedicated `Chat` button.

### ⌨️ Function & Hotkey Panel
* Essential PC keys are grouped in the top left corner for quick access:
  * `ESC` — menu / cancel.
  * `F1` — toggle in-game HUD.
  * `F3` — debug screen (coordinates and technical info).
  * `F5` — quick camera view toggle (1st/3rd person).
  * `E` — quick inventory access.

### 📝 Branding & Versioning:
* The interface is now officially labeled with the current build: `V.1.8.PRE-Release`.
* Author's signature added to the bottom panel: `by Brokens2W`.
